My Beamer Template
==================

Requirements
------------

- MacTeX (TeXLive?)

```sh
$ brew cask install mactex
```

Fonts
-----

- Helvetica
- DejaVu Sans Mono
- Hiragino Kaku Gothic ProN

Create new slide
----------------

```sh
$ git clone git://github.com/amutake/beamer-template.git new-slide
$ cd new-slide
$ make cont
```

Enjoy!
